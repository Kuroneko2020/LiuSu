# Phase-7 进度说明

## PPI 设置状态
- 设置页当前暴露的是自动拼版默认 PPI，可在 300、600、自定义之间切换，并通过 `auto/defaultPpi` 持久化。
- 导出页仍提供当前导出任务的 300、600、自定义 PPI；`export/defaultCustomPpi` 有持久化字段，但设置页没有单独暴露“默认自定义导出 PPI”输入。
- 原图导出模式不使用 PPI 决定像素尺寸，但导出文件会按实际页面物理尺寸写入分辨率元数据。

## 新任务确认逻辑
- 首页发起新任务时，若当前项目已有有效页面，先弹确认对话。
- 用户确认后才真正重置会话；取消则保持原项目不变。

## 胶片栏队列收紧
- `FilmStrip.qml` 当前以 `appController.project.pageCount` 为 model，显示所有页面。
- 页面缩略图会按页面状态刷新，支持横向滚动、拖动浏览与点击切换。
- “仅显示有效页面 + 当前草稿页”的队列折叠策略尚未在当前代码中实现。

## 基础缓存治理
- 启动时按设置清理标准化缓存、缩略图缓存、旋转/镜像派生缓存和槽位预览缓存中过旧文件。
- 当前默认保留时间为 30 天，设置页可选 1 天、1 周、1 个月。
- 在不引入复杂缓存数据库的前提下，先控制目录增长。

## 仍留到下一阶段
- 新任务确认交互文案与多语言细化。
- 胶片栏队列折叠策略（例如仅显示有效页 + 当前草稿页，或按最后编辑时间收敛）。
- 缓存容量上限与主动清理入口。

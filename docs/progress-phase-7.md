# Phase-7 进度说明

## 默认自定义 PPI 闭环
- 设置页新增默认自定义 PPI 输入。
- 该值已通过 QSettings 持久化并在启动恢复。
- 当默认分辨率为“自定义 PPI”时，导出页与自动模式都使用该默认值。

## 新任务确认逻辑
- 首页发起新任务时，若当前项目已有有效页面，先弹确认对话。
- 用户确认后才真正重置会话；取消则保持原项目不变。

## 胶片栏队列收紧
- 胶片栏改为显示“有效页面 + 当前草稿页”。
- 避免长期堆积大量空白页。

## 基础缓存治理
- 启动时清理标准化缓存和缩略图缓存目录中过旧文件（当前策略：14 天）。
- 在不引入复杂缓存数据库的前提下，先控制目录增长。

## 仍留到下一阶段
- 新任务确认交互文案与多语言细化。
- 队列折叠策略更细粒度（例如按最后编辑时间）。
- 缓存容量上限与主动清理入口。

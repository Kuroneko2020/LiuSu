# 实现计划（当前代码状态）

## 1. 本阶段目标
聚焦设置闭环、容错与预览/导出一致性收敛：
- 自动拼版默认 PPI 与导出任务 PPI
- 新任务防误触确认
- 胶片栏缩略图稳定刷新
- 自动策略可维护性继续提升
- 基础缓存治理

## 2. 关键实现

### 2.1 PPI 设置
- 设置页暴露自动拼版默认 PPI，支持 300、600、自定义。
- QSettings 持久化 `auto/defaultPpi` 与导出上下文的 `export/defaultCustomPpi`。
- 导出页可为当前导出任务选择 300、600、自定义 PPI。
- 原图导出不按 PPI 计算像素尺寸，但保存时按页面物理尺寸写入分辨率元数据。

### 2.2 新任务确认
- 首页触发新任务时，若项目中有有效页，先发确认请求。
- 用户确认后才执行 `startNewSession()`。

### 2.3 胶片栏逻辑
- `FilmStrip.qml` 当前显示所有页面，model 来自 `appController.project.pageCount`。
- 胶片栏通过页面缩略图、缩略图修订号与缓存失效信号保持刷新。
- “仅显示有效页 + 当前草稿页”的队列折叠策略仍是后续待做。

### 2.4 自动策略收紧
- `AutoLayoutPolicy` 继续收敛，关键阈值提为常量。
- AppController 仅保留流程编排。

### 2.5 缓存治理
- 应用启动时清理缓存目录中过旧文件（标准预览、缩略图、旋转/镜像派生、槽位预览）。
- 手动清理缓存、切换缓存目录、修改预览质量后，会先清理旧输出，再刷新当前项目资源并通知缩略图/预览失效。
- 当前采用轻量时间窗口策略，避免无限增长。

## 3. 本阶段不做
- 自定义模板编辑器
- 更多纸型/打印机预设
- AI 智能裁切
- 云同步、多图层、滤镜调色
- RAW/PSD/PDF、HEIC/HEIF 完整稳定支持

## 4. 限制说明
- 当前容器缺 Qt6 SDK，无法完整编译与交互验证。
- 缓存治理目前是轻量策略，未引入容量上限数据库。

## 5. 下一阶段建议（Phase-8）
1. 队列空白页折叠策略细化。
2. 新任务确认交互细节增强。
3. 缓存容量上限。
4. 自动策略参数化与可视化配置。

## 2. Phase-3 实施重点

### 2.1 真实导出链路
- `ExportService` 负责从 `ProjectState` 读取页面与槽位状态。
- 按 6x4 英寸（3:2）与 PPI（300/600/自定义）计算输出像素尺寸。
- 渲染页内容时应用：
  - 槽位图片
  - 旋转
  - 镜像
  - 填充模式（完整放入/铺满裁切）
- 支持 JPG / PNG 导出，支持当前页与全队列导出。

### 2.2 导出参数可用化
- 导出页可选路径、格式、命名规则、分辨率、裁切标记。
- “导出当前页”进入导出页时默认范围=当前页。
- “导出全队列”进入导出页时默认范围=全队列。
- 导出成功/失败提供明确消息。

### 2.3 文件名规则
- 组合命名：由页面中参与排版的原始文件名组合，过滤非法字符并限长。
- 日期-序号：`yyyyMMdd_001` 递增。
- 冲突处理：自动追加后缀，避免覆盖既有文件。

### 2.4 裁切标记
- 导出时可选绘制简化裁切标记（默认关）。
- 标记放置在页面与槽位边界的辅助位置。

### 2.5 导出相关设置持久化
- 用 `QSettings` 保存/恢复：
  - 默认导出路径
  - 记忆上次路径
  - 默认导出格式
  - 默认导出分辨率
  - 默认裁切标记

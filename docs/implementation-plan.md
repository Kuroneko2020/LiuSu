# 实现计划（更新到 Phase-7）

## 1. 本阶段目标
聚焦设置闭环、容错与队列体验收敛：
- 默认自定义 PPI 全链路
- 新任务防误触确认
- 胶片栏队列收紧
- 自动策略可维护性继续提升
- 基础缓存治理

## 2. 关键实现

### 2.1 默认自定义 PPI
- 设置页新增“默认自定义 PPI”输入。
- QSettings 新增 `export/defaultCustomPpi` 持久化。
- 启动恢复后同步到导出上下文 `customPpi`。
- 自动模式/导出页共用该默认值。

### 2.2 新任务确认
- 首页触发新任务时，若项目中有有效页，先发确认请求。
- 用户确认后才执行 `startNewSession()`。

### 2.3 胶片栏队列逻辑
- `ProjectState` 提供可见页集合逻辑：仅显示有效页 + 当前草稿页。
- 胶片栏改用可见页索引映射，降低空白页堆积。

### 2.4 自动策略收紧
- `AutoLayoutPolicy` 继续收敛，关键阈值提为常量。
- AppController 仅保留流程编排。

### 2.5 缓存治理
- 应用启动时清理缓存目录中过旧文件（标准化缓存 + 缩略图缓存）。
- 当前采用轻量时间窗口策略，避免无限增长。

## 3. 本阶段不做
- 自定义模板编辑器
- 更多纸型/打印机预设
- AI 智能裁切
- 云同步、多图层、滤镜调色
- RAW/PSD/PDF、HEIC/HEIF 完整稳定支持

## 4. 限制说明
- 当前容器缺 Qt6 SDK，无法完整编译与交互验证。
- 缓存治理目前是轻量策略，未引入容量上限数据库。

## 5. 下一阶段建议（Phase-8）
1. 队列空白页折叠策略细化。
2. 新任务确认交互细节增强。
3. 缓存容量上限与手动清理入口。
4. 自动策略参数化与可视化配置。

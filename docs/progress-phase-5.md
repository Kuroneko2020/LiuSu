# Phase-5 进度说明

## 本次修复的一致性问题
- 首页重新选模板会开启新任务，不再沿用旧页面。
- 自动模式预设已接入真实规则，不再只是字符串。
- 预览与导出在 fill-crop 偏移模型上统一。
- 槽位替换新图时不再继承旧图变换状态。
- 胶片缩略图加入版本化 URL，稳定刷新。

## 自动模式预设当前规则
- 均衡填充：按宽高比差异决定 Fit/Crop。
- 人像优先：横图优先旋转为竖向，默认 Crop。
- 证件照优先：默认 Fit。

## 预览/缩略图/导出如何共享逻辑
- 模板布局统一来自 `TemplateLayout`。
- 图片基础统一来自 `ImageService` 标准化缓存。
- 缩略图由 `ExportService` 渲染链路生成。
- fill-crop 偏移采用统一归一化模型并写入 `ProjectState`。

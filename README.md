# Photo Template Editor

这是一个桌面端的照片拼版小工具，目标是帮你把多张照片快速排成 6 英寸相纸页面，方便后续统一导出。  
项目基于 **Qt 6 + C++ + QML** 开发，当前版本已经有可运行的主流程，可以继续往“真正可生产使用”的方向迭代。

---

## 这个项目是干啥的？

一句话：**把“选图 -> 排版 -> 调整 -> 导出”这套流程做成一个轻量桌面工具**。

目前主要面向这种场景：

- 需要把多张照片拼成固定页面（比如 2 宫格 / 4 宫格 / 9 宫格）  
- 希望快速批量处理，不想每次都去设计软件里手工摆  
- 需要导出成可打印或数字发布的图片

---

## 现在有什么功能？

当前已实现（持续完善中）：

- **首页任务入口**
  - 手动创建排版任务
  - 自动模式导入图片后按规则分配槽位

- **编辑页**
  - 槽位选图、替换、删除
  - 旋转、镜像、填充模式切换、构图偏移
  - 页面背景（纯色 / 纹理）设置
  - 自定义取色器（应用内弹窗）

- **胶片栏**
  - 页面缩略图浏览与切换
  - 横向滚动与拖动浏览（多页时）

- **导出页**
  - 导出当前页 / 导出全部
  - 命名规则、格式、PPI、裁切标记
  - 原图导出模式（开启后自动 PNG 并忽略 PPI）

- **设置页**
  - 自动拼版默认参数
  - 缓存目录、缓存保留时间
  - 纹理目录设置与管理

- **UI**: QML / Qt Quick / Qt Quick Controls 2

- **关键模块**: ProjectState / AppController / ImageService / ExportService / AutoLayoutPolicy

## 快速启动

### 依赖

需要安装 Qt 6.5+（至少包含以下模块）：

- Core
- Gui
- Qml
- Quick
- QuickControls2

### 构建运行

```bash
cmake -S . -B build
cmake --build build
./build/photo_template_editor
```

如果 CMake 提示找不到 `Qt6Config.cmake`，通常是本机 Qt SDK 没装全，或者 `CMAKE_PREFIX_PATH` / `Qt6_DIR` 没配好。
